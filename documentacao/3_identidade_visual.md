# Identidade Visual - ALFABetiza

## 1. Conceito Visual
O design deve transmitir **clareza**, **acessibilidade** e **alegria**. Deve ser convidativo para crianças, mas respeitoso o suficiente para adultos (evitando excesso de elementos "bebês").

## 2. Paleta de Cores
Cores vibrantes para engajamento, com alto contraste para acessibilidade.

- **Primária (Ação):** Azul Real (`#2563EB`) - Confiança, calma, aprendizado.
- **Secundária (Destaque):** Amarelo Ouro (`#F59E0B`) - Energia, atenção, gamificação.
- **Sucesso:** Verde Esmeralda (`#10B981`) - Acerto, progresso.
- **Erro/Atenção:** Vermelho Suave (`#EF4444`) - Correção.
- **Fundo:** Off-White (`#F8FAFC`) ou Creme Suave (`#FFFBEB`) - Menos cansativo que branco puro.
- **Texto:** Cinza Escuro (`#1E293B`) - Melhor que preto puro para leitura.

## 3. Tipografia
Foco total na legibilidade.

- **Família Principal:** `Inter` ou `Poppins` (Google Fonts). São modernas, geométricas e altamente legíveis.
- **Tamanhos:**
  - Títulos: Grande e Bold (ex: 32px+).
  - Corpo: Mínimo 18px para facilitar a leitura.
  - Espaçamento: Entrelinha generosa (1.5 ou 1.6).

## 4. Iconografia
- Ícones sólidos e simples (estilo Phosphor ou Heroicons).
- Traços espessos (2px ou 3px) para fácil reconhecimento.
- Uso de metáforas literais (ex: Casa para início, Lápis para exercícios).

## 5. Elementos de UI
- **Botões:** Grandes, com cantos arredondados (raio de 12px ou pill-shape). Devem parecer "clicáveis".
- **Cards:** Uso de sombras suaves para dar profundidade e separar conteúdo.
- **Imagens:** Ilustrações vetoriais planas (flat design) ou semi-realistas. Evitar fotos genéricas de banco de imagens.

## 6. Acessibilidade (Diretrizes WCAG)
- Contraste mínimo de 4.5:1 entre texto e fundo.
- Suporte a redimensionamento de texto dinâmico.
- Áreas de toque de no mínimo 48x48px.
